## Repository Layout

- `hackathon-starter/`: Next.js application, shared starter docs, and AI guidance.
- `.github/`: Repository-level GitHub templates for issues and pull requests.

Use the starter directory as the application root when installing dependencies, running scripts, and configuring environment variables.
